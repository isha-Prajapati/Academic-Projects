# Analysis of WFH Population Data in Ireland

A statistical data analysis project exploring the impact of COVID-19 on remote work culture in Ireland using descriptive and inferential statistical techniques.

---

## Project Overview

This project analyzes the growth and distribution of Work From Home (WFH) trends across Ireland using population and labor force data. The analysis focuses on how the COVID-19 pandemic transformed remote working patterns across industries, regions, and gender groups.

The project uses statistical methods to identify trends, compare demographic shifts, and validate findings through hypothesis testing and ANOVA analysis.

---

## Objectives

- Analyze the growth of remote work in Ireland
- Study occupational groups with the highest WFH adoption
- Compare regional differences in remote working trends
- Examine gender-based changes in remote work participation
- Apply statistical testing techniques to validate findings

---

## Dataset

The dataset used in this project contains population and employment statistics related to remote work in Ireland for the years:

- 2011
- 2016
- 2022

Data includes:
- Region-wise WFH population
- Occupation-wise remote workers
- Gender distribution
- Labor force statistics

Data Source:
- Central Statistics Office (CSO) Ireland

## Key Findings

### Remote Work Growth
- Remote work adoption increased significantly after COVID-19.
- The percentage of employees working remotely more than doubled in 2022 compared to previous years.

### Industries with Highest WFH Adoption
The following industries showed the strongest shift toward remote work:
- Information & Communication
- Professional, Scientific & Technical Activities
- Financial & Insurance Activities

### Regional Trends
- Dublin and Cork experienced the highest increase in remote workers.
- Eastern and Southern regions showed stronger WFH adoption than other regions.

### Gender Analysis
- Female participation in remote work increased significantly post-COVID.
- Statistical testing confirmed that the increase was significant at the 5% significance level.

### ANOVA Findings
- Significant regional differences were identified in WFH population trends across Ireland.
